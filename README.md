## Tentang Saya

Mahasiswa **Ekonomi Islam** di UIN Sjech M. Djamil Djambek Bukittinggi — tapi rasa ingin tahu saya tidak berhenti di sana.

Saya termasuk orang yang tidak bisa membiarkan perangkat berjalan apa adanya. Setiap HP yang saya pegang sudah di-root dan dikustomisasi. Windows saya pakai versi custom. Laptop saya dual-boot Ubuntu. Bukan karena profesi, tapi karena penasaran — dan penasaran itu yang terus mendorong saya belajar hal-hal baru.

Tidak ada roadmap khusus. Saya belajar apa yang menarik perhatian saya, dan selama ini itu sudah membawa saya cukup jauh.
